########################
OpenERP 第一步
########################

.. i18n:  OpenERP is an impressive software system, being easy to use and yet providing great benefits
.. i18n:  in helping you manage your company.
.. i18n:  It is easy to install under both Windows and Linux compared to other enterprise-scale systems,
.. i18n:  and offers unmatched functionality.
..

 OpenERP是一个非常棒的应用系统，它容易上手操作而且可以很好地帮助你管理公司。
 相比其他企业级的应用软件，它可以很简单地安装在Windows和Linux操作系统上，并提供很多实用的功能。

.. i18n:  The objective of this first part of the book is to help you start discovering OpenERP in practice.
..

 本书第一章的目的是帮助你开始在练习中了解OpenERP。

.. i18n:  The first chapter, :ref:`ch-inst`, gives detailed guidance for installing it.
.. i18n:  If you are not a system administrator, or if you have already installed
.. i18n:  OpenERP, or if you are planning to use an online SaaS provider, then you can skip this chapter and
.. i18n:  move straight to the next chapter, :ref:`ch-guided`. 
.. i18n:  There we take you on a step-by-step guided tour using the information 
.. i18n:  in the demonstration database.
.. i18n:  
.. i18n:  If you have already used OpenERP (or Tiny ERP) a bit then you can move to the third chapter in this part of the book.
.. i18n:  In :ref:`ch-real` you can try out a real case, from scratch in a new database,
.. i18n:  by developing a complete business workflow that runs from purchase to sale of goods.
.. i18n:  
.. i18n:  .. figure:: images/sect1_outline.png
.. i18n:    :align: center
.. i18n: 
.. i18n:    *Options for reading this part of the book*
..

 第一节, :ref:`ch-inst` , OpenERP的详细安装配置指导.
 如果你不是系统管理员，或者你已经安装好OpenERP，或者你打算选择在线SaaS服务商，那么你可以跳过这章，阅读下一节  :ref:`ch-guided` 。
 我们会一步一步地带领你浏览使用demo数据。
 
 如果你曾经使用过OpenERP（或者Tiny ERP），那么你可以直接阅读第三节。
 在 :ref:`ch-real` ，你可以尝试根据实际案例创建一个新的帐套，并模拟运行一个从采购到销售的完整业务流程。

   :align: center

   *如何选择阅读本书第一章的内容*
